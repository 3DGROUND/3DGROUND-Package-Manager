## ğŸ“ How Package Installation Works

The manager does **not** install the files manually.  
Instead, it performs the following automated sequence:

1. Downloads the official MZP installer  
2. Extracts the archive  
3. Runs the MZP installer in **silent mode**  
4. The package installer itself:
   - copies files  
   - places assets  
   - installs icons  
   - creates Toolbar buttons  
   - creates Quad Menu items  

Everything happens invisibly and requires **zero interaction** from the user.
